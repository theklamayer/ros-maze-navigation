# ROS Maze Navigation Robot – Team Project
This repository is a showcase version of a robotics project from TU Berlin (Robotics Fundamentals, 2024).

The original repository was developed collaboratively and is private. This public version is intended to document the project structure, technologies used, and implementation approach for portfolio/recruiting purposes.

The robot can:
- navigate a grid maze
- localize itself using sensor data
- plan paths to target positions

My contributions:
- Implemented PD-controlled movement for improved accuracy  
- Developed corridor detection to optimize movement  
- Implemented BFS path planning
- Contributed to system integration
